# 🌐 Local Service Provider Platform

## 📌 Project Description

This project is a **Local Service Provider Platform** where users can find and book services like electrician, plumber, and cleaner.

The main aim of this project is to make it easy for users to connect with nearby service providers and book services online.

## 🎯 Features

* User Registration and Login
* Book Local Services
* Simple Dashboard
* Feedback System
* Easy to use interface

## 🛠️ Technologies Used

* HTML (Structure of website)
* CSS (Design and styling)
